# AgroPredict - Inteligência de Dados na Soja 🚜📈

> **Status do Projeto:** Em fase de Planejamento Estratégico e Arquitetura de Dados.

## 🌟 Visão do Projeto
Este projeto nasce da convergência entre a expertise em Auditoria de Negócios e as tecnologias de ponta em **IA e IoT**. O objetivo é transformar a volatilidade do mercado de Soja em previsibilidade de lucro para o produtor rural.

## 🚀 O que está por vir?
Estamos construindo um modelo preditivo que não olha apenas para o gráfico de preços, mas para o ecossistema global:
- **Variáveis Exógenas:** Impacto do Dólar, Petróleo (Frete) e Geopolítica.
- **Inteligência Climática:** Correlação de safras globais (EUA/BR).
- **IoT no Campo:** Integração de telemetria para apuração de custo real.

## 🎓 Contexto Acadêmico
Desenvolvido como projeto prático durante a especialização em **Ciência de Dados e IoT (OpAIoT)** no **ICMC-USP São Carlos**.
